วิธีใช้งาน

1) แตกไฟล์ zip
2) เข้าโฟลเดอร์โปรเจกต์
3) ติดตั้งแพ็กเกจ
pip install -r requirements.txt

4) รัน
python3 app.py

5) เปิดเว็บ
http://127.0.0.1:8000

แอดมินเริ่มต้น
username: admin
password: 1234

ความสามารถเพิ่มจากรอบนี้
- กำหนดค่าสมัครแต่ละอีเวนต์
- ผู้สมัครแนบสลิปค่าสมัครได้
- แอดมินเปิดดูสลิปได้
- รองรับไฟล์ jpg, jpeg, png, webp, pdf
